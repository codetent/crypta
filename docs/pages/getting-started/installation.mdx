# Installation

Crypta consists only of one binary without any dependencies.

<Tabs
  defaultValue="windows"
  values={[
    { label: 'Windows', value: 'windows', },
    { label: 'Darwin', value: 'darwin', },
    { label: 'Linux', value: 'linux', },
  ]}
>
<TabItem value="windows">

```powershell
curl -L -o crypta.exe "https://github.com/codetent/crypta/releases/latest/download/crypta_Windows_x86_64.exe"
```

</TabItem>
<TabItem value="darwin">

```bash
curl -L -o crypta "https://github.com/codetent/crypta/releases/latest/download/crypta_Darwin_x86_64" && sudo install -c -m 0755 crypta /usr/local/bin
```

</TabItem>
<TabItem value="linux">

```bash
curl -L -o crypta "https://github.com/codetent/crypta/releases/latest/download/crypta_Linux_x86_64" && sudo install -c -m 0755 crypta /usr/local/bin
```

</TabItem>
</Tabs>
